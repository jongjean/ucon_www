─────────────────────────────────────────────
1. 프로젝트 개요
─────────────────────────────────────────────
UCONCREATIVE 주식회사의 공식 웹사이트 개발 프로젝트로,
AI 서비스 브랜드 **UCONAI** 를 중심으로 한 기업 홍보 및 기술 소개를 목표로 함.

본 프로젝트는 로컬 개발 환경(Laragon)과 GitHub을 완전 연동하여,
로컬에서 개발 → GitHub으로 버전 관리 → DDNS 기반 서버로 실시간 반영되는
**통합 웹개발 파이프라인**을 구축함.

─────────────────────────────────────────────
2. 주요 기술 스택
─────────────────────────────────────────────
- WordPress 6.x
- PHP 8.x
- MySQL 8.x
- Apache (Laragon 기본 환경)
- Git + GitHub (버전 관리)
- Visual Studio Code (코드 작성 및 테마 개발)
- Astra 테마 + Astra Child 테마 커스터마이징
- DDNS (uconcreative.ddns.net) 기반 외부 공개

5. 주요 기술 포인트
─────────────────────────────────────────────
- GitHub 자동 버전 관리
  → VS Code에서 커밋 시 GitHub와 즉시 동기화
- DDNS 기반 원격 접속
  → uconcreative.ddns.net으로 외부 접근 가능
- Child Theme 구조
  → astra-child/functions.php 를 통해 기능 확장
- 커스텀 숏코드
  → [ucon_news_cards], [ucon_products] 등 내부 컴포넌트화
- AI + UI 통합 기반 구조
  → 추후 AI 기반 콘텐츠 생성 플러그인 연계 예정
